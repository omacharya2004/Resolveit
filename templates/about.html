PingMe

A sleek, real-time chat app with room codes.

What is PingMe?

PingMe is a lightweight real-time chat application built with Python, Flask, and Socket.IO. Create or join a chat using a simple room code and start messaging instantly. It supports multi-room chats, online user presence, private messages, and an elegant dark UI.

Features

  • Room code based chats
  • Live messaging via WebSockets
  • User presence and join/leave notifications
  • Private messages between users
  • Optional SQLite persistence
